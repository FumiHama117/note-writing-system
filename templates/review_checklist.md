# 投稿前チェックリスト

## 必須チェック（全項目クリアで投稿OK）

### タイトル
- [ ] 読者のベネフィットが1文で伝わるか
- [ ] 数字または具体的な言葉が入っているか
- [ ] 28文字以内に収まっているか
- [ ] クリックしたくなるか（自分で確認）

### リード文
- [ ] 最初の1文で読者の悩みを言語化できているか
- [ ] 「これ自分のことだ」と感じさせる具体性があるか
- [ ] この記事を読むと何が得られるか明示されているか
- [ ] 200字以内に収まっているか

### 本文
- [ ] H2が3〜5個あるか
- [ ] 各H2にH3が2〜3個あるか
- [ ] 具体的な数字が含まれているか
- [ ] 体験談・実例が含まれているか
- [ ] 1段落が5行以内に収まっているか
- [ ] 箇条書きが適切に使われているか（詰め込みすぎていないか）

### 締め
- [ ] 本文の要約（3点以内）があるか
- [ ] 次の行動を1つだけ明示しているか
- [ ] スキ・フォローへの自然な呼びかけがあるか

### 全体
- [ ] 文字数が3,000〜5,000字の範囲内か
- [ ] 誤字脱字がないか（音読して確認）
- [ ] ハッシュタグを3〜5個設定したか
- [ ] サムネイルを設定したか
- [ ] 有料/無料の設定を確認したか

## 品質向上チェック（できれば対応）

- [ ] 図・表・引用ブロックを適切に使っているか
- [ ] 内部リンクを3個以上入れているか
- [ ] SNSシェア用のコピーを準備したか
- [ ] 投稿タイミングは最適か（水・木・日の20〜22時推奨）

## 投稿後チェック

- [ ] SNS（X/Twitter等）でシェアしたか
- [ ] 投稿後1時間のPV・スキ数を確認したか
- [ ] コメントへの返信をしたか
