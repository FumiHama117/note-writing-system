# 最初の3行テンプレ

## リード文の目的

最初の3行で「これ自分のことだ」と思わせ、最後まで読み続けてもらう。

## 黄金テンプレート

### テンプレート1: 共感 → 問題提起 → 解決予告
```
〇〇しているのに、なぜか△△できない——そんな経験はありませんか？

私も同じ悩みを抱えていました。[具体的な失敗談・状況]

この記事では、[解決策の核心]を具体的なステップでお伝えします。読み終えると、[読者が得られるもの]ができるようになります。
```

**例**:
> 毎日noteを更新しているのに、フォロワーが全然増えない——そんな経験はありませんか？
>
> 私も3ヶ月間、毎日投稿を続けたのにフォロワーが50人しか増えませんでした。
>
> この記事では、投稿頻度より大切な「読まれる仕組み」を5つのステップで解説します。実践すれば、月に300人以上のフォロワーを獲得できるようになります。

---

### テンプレート2: 数字インパクト → 問い → 答えの予告
```
[驚きの数字・事実]

なぜ〇〇な人が△△できるのか、その答えはシンプルです。

この記事では、[具体的な内容]をお伝えします。
```

**例**:
> noteで月100万円以上稼ぐクリエイターの9割が、ある共通点を持っています。
>
> なぜフォロワー数が少ないのに高収益を出せるのか、その答えはシンプルです。
>
> この記事では、トップクリエイター20人を分析して見えてきた「収益化の法則」を全公開します。

---

### テンプレート3: 悩み直撃 → 共感 → 逆転の提示
```
〇〇で悩んでいませんか？

実はその悩み、[意外な原因]が原因かもしれません。

[解決策]を知るだけで、状況は一変します。この記事でその全てをお伝えします。
```

## リード文チェックリスト

- [ ] 最初の1文で読者の悩みを言語化できているか
- [ ] 「自分のことだ」と感じさせる具体性があるか
- [ ] この記事を読むと何が得られるか明示されているか
- [ ] 200字以内に収まっているか（長すぎると離脱される）
- [ ] 「です・ます」の敬体で統一されているか
