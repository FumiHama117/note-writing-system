# サムネイル設計

## サムネイルの重要性

noteのタイムラインでは、サムネイルが第一印象を決める。クリック率に直結する最重要要素。

## バズるサムネイルの法則

### 視覚デザイン
- **背景色**: 単色または2色まで（白・黒・青・オレンジが人気）
- **フォント**: 太字・明朝体またはゴシック体
- **文字数**: タイトルは20文字以内に圧縮してサムネに載せる
- **余白**: 詰め込みすぎない（全体の30%は余白）

### 効果的なサムネイルパターン

| パターン | 説明 | 向いているジャンル |
|----------|------|-------------------|
| テキストのみ | キャッチコピー + 大きな数字 | ノウハウ・ビジネス |
| 人物写真 + テキスト | 著者顔写真で信頼感 | 体験談・自己啓発 |
| 図解・インフォグラフィック | 内容の一部を可視化 | 解説・比較記事 |
| イラスト + テキスト | 親しみやすさ | 日常・エッセイ |

## 作成ツール

- **Canva**: テンプレート豊富、無料で使える
- **Figma**: デザイン精度高い、慣れが必要

## サムネイルチェックリスト

- [ ] スマホ画面（小サイズ）で文字が読めるか
- [ ] 一瞬で何の記事か伝わるか
- [ ] 色のコントラストが十分か
- [ ] 他の記事と差別化できているか
