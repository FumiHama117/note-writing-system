# H2/H3の黄金比

## 基本構成ルール

- **H2**: 3〜5個（記事の大きな章）
- **H3**: 各H2に2〜3個（章の中の小見出し）
- **H4以下**: 原則使わない（視認性が落ちる）

## 黄金構成パターン

### パターンA: 問題解決型（3,000字）
```
## 1. 〇〇が上手くいかない本当の理由
  ### 1-1. よくある誤解
  ### 1-2. 根本的な原因

## 2. 解決するための3ステップ
  ### 2-1. ステップ1：〇〇
  ### 2-2. ステップ2：△△
  ### 2-3. ステップ3：□□

## 3. 実践するためのポイント
  ### 3-1. よくある失敗と対策
  ### 3-2. 継続するためのコツ
```

### パターンB: ノウハウ集型（4,000字）
```
## 1. まず知っておきたい前提
  ### 1-1. 〇〇とは何か
  ### 1-2. なぜ重要なのか

## 2. 具体的な方法N選
  ### 2-1. 方法1
  ### 2-2. 方法2
  ### 2-3. 方法3

## 3. 応用編：さらに効果を上げるには
  ### 3-1. 上級テクニック
  ### 3-2. ツール活用法

## 4. 実践スケジュール
  ### 4-1. 最初の1週間
  ### 4-2. 1ヶ月後の目標
```

### パターンC: 体験談型（5,000字）
```
## 1. 始める前の私の状況
  ### 1-1. 抱えていた悩み
  ### 1-2. きっかけ

## 2. 試したこと・失敗したこと
  ### 2-1. 最初の失敗
  ### 2-2. 転換点

## 3. うまくいった方法
  ### 3-1. 具体的な手順
  ### 3-2. 数字で見る成果

## 4. あなたへのアドバイス
  ### 4-1. まず最初にやること
  ### 4-2. やってはいけないこと

## 5. まとめ
```

## 見出しの書き方ルール

1. **体言止めより動詞で終わる** → 「フォロワーを増やす方法」より「フォロワーを3倍に増やす3つの方法」
2. **各H2で完結感を持たせる** → 読み飛ばしても内容が推測できる
3. **数字・具体性を入れる** → 「いくつかのコツ」より「5つのコツ」
