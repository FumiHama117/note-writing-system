# バズるタイトルの型集

## 基本原則

1. 読者のベネフィットが1文で伝わること
2. 数字・固有名詞・具体性を入れること
3. 28文字以内を目安に（スマホでの視認性）

## タイトルの型一覧

### 型1: 数字 + ベネフィット型
```
【テンプレート】
〇〇するための「N個」の方法
N日で〇〇できた具体的な手順
〇〇が変わるN個の習慣
```
**例**: 「フォロワー0から1,000人になった5つの戦略」

---

### 型2: ターゲット明示型
```
【テンプレート】
〇〇な人のための△△
〇〇している人が△△できる理由
〇〇で悩む人へ：△△の解決策
```
**例**: 「文章が苦手な人でも月5万円稼げるnote術」

---

### 型3: 逆説・意外性型
```
【テンプレート】
なぜ〇〇しても△△できないのか
〇〇してはいけない理由
実は〇〇が△△を妨げていた
```
**例**: 「毎日投稿してもフォロワーが増えない本当の理由」

---

### 型4: How to + 具体的結果型
```
【テンプレート】
〇〇を△△するだけで□□が変わる
〇〇するたった1つの方法
〇〇が劇的に変わる△△のやり方
```
**例**: 「タイトルを変えるだけでPVが3倍になった話」

---

### 型5: 実体験・告白型
```
【テンプレート】
〇〇が△△だった私が□□できた理由
〇〇して分かった△△の真実
〇〇年間〇〇してみて気づいたこと
```
**例**: 「3ヶ月でnoteを辞めた私が再開して月10万円達成した話」

---

### 型6: 完全ガイド型
```
【テンプレート】
〇〇の完全ガイド【初心者向け】
〇〇の始め方：基礎から応用まで全解説
〇〇を徹底解説：初心者が最初に読む記事
```
**例**: 「noteの始め方【2024年版】完全ガイド」

## タイトル作成チェックリスト

- [ ] 読者のベネフィットが明確か
- [ ] 数字または具体的な言葉が入っているか
- [ ] 誰に向けた記事か分かるか
- [ ] 読んだ後の状態が想像できるか
- [ ] クリックしたくなるか（自分で確認）
