# 締めの一文

## 締め文の役割

記事の最後に読者を「次のアクション」に向かわせる。スキ・フォロー・購入の意思決定に影響する最重要パート。

## 締め文の型

### 型1: アクション促進型（最もよく使う）
```
今すぐ[具体的な行動]してみてください。小さな一歩が、大きな変化を生み出します。
```
**例**: 「今すぐnoteのプロフィールを見直してみてください。たった5分の作業が、フォロワー獲得に直結します。」

---

### 型2: 問いかけ型（エンゲージメント狙い）
```
あなたは[問い]について、どう思いますか？コメントで教えていただけると嬉しいです。
```
**例**: 「あなたがnoteを続ける上で一番難しいことは何ですか？コメントで教えていただけると嬉しいです。」

---

### 型3: 次回予告型（継続読者獲得）
```
次回は[次の記事のテーマ]について書きます。見逃したくない方はフォローをお願いします。
```

---

### 型4: 感謝 + 呼びかけ型（コミュニティ感）
```
最後まで読んでいただきありがとうございました。この記事が少しでもお役に立てたなら、スキ・シェアしていただけると嬉しいです。
```

## 絶対に避けること

- 「以上です」で終わる（余韻がない）
- スキ・フォローを強要する文言（反感を買う）
- まとめだけで終わる（アクションへの動線がない）
