# ポジショニング

## 差別化の軸

### ポジショニングマップ
```
            実践的・具体的
                  ↑
  初心者向け ←——————————→ 上級者向け
                  ↓
            理論的・抽象的
```

**狙うポジション**: 初心者向け × 実践的・具体的

### USP（独自の強み）
1. **再現性の高さ**: 手順通りやれば誰でも同じ結果を出せる
2. **失敗談の豊富さ**: 成功談だけでなく失敗からの学びを共有
3. **最新情報の反映**: noteのアップデートに合わせた最新ノウハウ

## ブランドボイス

- **トーン**: 親しみやすい、背中を押す、正直
- **文体**: 敬体（です・ます）、読者に語りかける
- **禁止**: 上から目線、過度な煽り、根拠のない断言

## コンテンツの柱（ピラーコンテンツ）

1. **始め方系**: note初心者が最初にやること
2. **継続系**: 書き続けるための仕組み作り
3. **マネタイズ系**: 有料記事・メンバーシップ活用法
4. **拡散系**: SNS連携・ハッシュタグ戦略
