# ハッシュタグ運用

## noteのハッシュタグの役割

- タグ検索からの流入を増やす
- 関連コンテスト・企画への参加
- 同ジャンルのクリエイターとの繋がり形成

## 効果的なハッシュタグ戦略

### 使用数の目安
- **推奨**: 3〜5個
- 多すぎると希薄化・スパム感が出る

### タグ選定の組み合わせ

| タグの種類 | 目的 | 例 |
|------------|------|-----|
| 大タグ（10万件以上） | 幅広い露出 | #note #副業 #ライティング |
| 中タグ（1〜10万件） | 適度な競合 | #note運営 #情報発信 |
| 小タグ（1万件未満） | ニッチで上位表示 | #noteの書き方 |

**推奨比率**: 大1 + 中2 + 小2

### ジャンル別おすすめタグ

**note・情報発信系**
```
#note #noteの書き方 #情報発信 #コンテンツ制作 #ブログ運営
```

**副業・ビジネス系**
```
#副業 #フリーランス #マネタイズ #稼ぎ方 #副収入
```

**ライティング・文章術系**
```
#ライティング #文章術 #コピーライティング #文章力
```

## ハッシュタグ活用の注意点

- 記事内容と無関係なタグは使わない（信頼性低下）
- 人気コンテスト・企画のタグは積極的に活用
- 定期的にタグの投稿数をチェックして更新する
